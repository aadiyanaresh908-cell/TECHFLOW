Semantic Structure: Used HTML5 tags like <header>, <nav>, <main>, and <article> to provide a clear content hierarchy for screen readers.

Performance Speed: Optimized load times by using CSS Variables to reduce code bloat and implementing loading="lazy" for images.

Visual Stability: Applied the aspect-ratio property to image containers to prevent Cumulative Layout Shift (CLS) during page load.

Keyboard Navigation: Integrated a "Skip to Main Content" link and visible focus states to support users who do not use a mouse.

Accessible Forms: Linked all <input> fields to explicit <label> tags and used aria-required to ensure a seamless experience for visually impaired users.

Color Contrast: Selected a high-contrast color palette (Indigo/White) that meets WCAG AA standards for readability.
